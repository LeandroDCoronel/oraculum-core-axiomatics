ORACULUM CORE AXIOMATICS

Oraculum is a personal cybernetic operating system designed to guide
human development through measurable daily actions.

Instead of vague productivity systems, Oraculum implements a structured
axiomatic architecture where human progress is tracked through domains,
KPIs, and missions.

The system transforms daily execution into a cybernetic feedback loop
between intention and action.


--------------------------------------------------
CORE CONCEPTS
--------------------------------------------------

DOMAINS

Human activity is divided into strategic domains.

Current domains:

A — Athletics
B — Business
T — Technology
S — Science
G — Gallery

Each domain contains KPIs (Key Performance Indicators) representing
concrete daily actions.


--------------------------------------------------
KPI SYSTEM
--------------------------------------------------

KPIs represent small but measurable actions.

Example:

A.KPI.1 — 7 km run completed
A.KPI.2 — mobility training
T.KPI.1 — code module implemented

Completion is recorded daily.


--------------------------------------------------
WEIGHTED PROGRESS
--------------------------------------------------

Domains have strategic weights.

Example:

Athletics   20%
Business    30%
Technology  30%
Science     10%
Gallery     10%

These weights generate a Global Performance Index.


--------------------------------------------------
OPERATIVE COST SYSTEM
--------------------------------------------------

Introduced in version v5.

Human development requires paying operative costs.

These costs appear as missions in two forms.


POSITIVE FRICTION

Constructive effort.

Example:

Mission 001 — Absolute Circadian Anchoring

Sleep before 21:00
Wake up 05:00
Morning run


NEGATIVE FRICTION

Elimination of destructive behaviors.

Example:

Mission 002 — Reorientation of Resource Flow

No weed
No alcohol
No compulsive scrolling
Financial discipline


--------------------------------------------------
HISTORIC ENGINE
--------------------------------------------------

Oraculum stores daily performance in JSON files.

Users can review:

Daily execution
Domain performance
Global progress
